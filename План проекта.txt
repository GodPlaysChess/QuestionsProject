Версия 1.0
Часть для ученика.
При заходе на сайт запрашивается фамилия, имя. - Одна форма.
Показывается случайный вопрос, с формой ответа. При нажатии кнопки отослать - ответ посылается на сервер и показывается следующий случайный вопрос.
Вопросы не могут повторятся. После того как задано N вопросов - показывается заглушка - экзамен завершен.

Часть для учителя.
Админская страница где выводятся все вопросы, и есть кнопка - добавить вопрос. Обычная textarea.
Админская страница со списком учеников. Клик по ученику открывает страницу с его ответами. Каждому ответу можно поставить оценку "+", "-".
Отдельная страница с результатом проверки - находится не в админской части, а в паблике.

Версия 2.0
Поддержка нескольких курсов. Добавление "веса" вопроса. Задание порогов суммы весов для получения конкретной оценки для каждого курса.
После логина ученик может выбрать какой курс он сдает, и вопросы задаются только из этого курса.

На странице преподавателя так же главной становится страница выбора курса.
Админская страница добавления новых преподавателей и учеников.
Ограничение попыток сдачи.

Версия 3.0
Проверка приватности.
Добавление различных типов вопросов. (Чекбокс, радиобаттон, с указанием правильных вариантов ответа и возможностью автоматической оценки без участия преподавателя).
Рич редактор для вопросов. 


Версия 4.0
Отсылка писем о завершении проверки с результатами на email ученика. Отсылка писем о проходжении экзамена на email преподавателя.
Добавление медиа (фото, видео, аудио) в вопрос.

Версия 5.0
Использование возможностей html5, оффлайн работа с приложением(для преподавателя).
